🗺️ ROUTES PRINCIPALES:

Publiques:
  GET /                               → Redirect /auth/login
  GET /public/urgence/:token          → Page urgence
  GET /public/ordonnance/:qr          → Vérif ordonnance

Authentification:
  GET  /auth/login                    → Page login ✅
  POST /auth/login                    → Soumission ✅ CORRIGÉ
  GET  /auth/logout                   → Déconnexion

Patient:
  GET /patient/dossier                → Dossier (lecture seule)
  GET /patient/ordonnances            → Liste ordonnances
  GET /patient/rdv                    → Rendez-vous
  GET /patient/examens                → Résultats examens

PDF Patient (NOUVEAU):
  GET /patient-pdf/ordonnance/:id     → PDF ordonnance ✅
  GET /patient-pdf/examen/:id         → PDF examen ✅

Médecin:
  GET  /medecin/consultations         → Liste consultations
  POST /medecin/nouvelle-consultation → Nouvelle consult
  POST /medecin/ordonnances/nouvelle  → Prescription

Agent accueil:
  GET  /accueil/nouveau-patient       → Créer patient
  POST /accueil/nouveau-patient       → Enregistrer
  GET  /accueil/rdv/nouveau           → Prendre RDV
  POST /accueil/rdv/nouveau           → Enregistrer RDV

Dashboards:
  GET /dashboard/admin                → Dashboard super admin
  GET /dashboard/structure            → Dashboard admin structure
  GET /dashboard/medecin              → Dashboard médecin
  GET /dashboard/pharmacien           → Dashboard pharmacien
  GET /dashboard/caissier             → Dashboard caissier
  GET /dashboard/accueil              → Dashboard accueil
  GET /dashboard/patient              → Dashboard patient
