## 🔍 DIAGNOSTIC DES ERREURS

### **Erreur : "Configuration du serveur incomplète"**
**Cause** : Variables d'environnement Cloudflare non configurées

**Solution** :
1. Aller sur https://dash.cloudflare.com/
2. Pages → `sante-bf` → Settings → Environment variables
3. Ajouter `SUPABASE_URL` et `SUPABASE_ANON_KEY`
4. Sauvegarder
5. Retry Deployment (ou push un nouveau commit)

### **Erreur : "Profil introuvable"**
**Cause** : Utilisateur existe dans `auth.users` mais pas dans `auth_profiles`

**Solution** :
```sql
-- Dans Supabase SQL Editor
INSERT INTO auth_profiles (id, nom, prenom, email, role, structure_id, est_actif)
VALUES (
  'uuid-de-l-utilisateur',  -- Récupérer depuis auth.users
  'Nom',
  'Prenom',
  'email@example.com',
  'super_admin',
  NULL,
  true
);
```

### **Erreur : "Lien expiré ou invalide" (reset password)**
**Vérifications** :
1. ✅ Le code a bien été déployé (vérifier commit `c95497a` dans Cloudflare)
2. ✅ Cache navigateur vidé (Ctrl+Shift+R)
3. ✅ Email de reset récent (<1h)
4. ✅ Redirect URL configurée dans Supabase : `https://sante-bf.pages.dev/auth/reset-confirm`

### **Erreur : Spinner infini**
**Vérifications** :
1. ✅ Variables env configurées
2. ✅ Supabase accessible (vérifier status : https://status.supabase.com/)
3. ✅ Logs Cloudflare consultés
4. ✅ Timeout de 30s devrait s'activer automatiquement
